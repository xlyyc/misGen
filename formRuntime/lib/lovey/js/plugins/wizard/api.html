插件方法详细
{jQuery} loveyInitWizard(op)
描述:
  • 步骤向导
参数列表:
参数名 类型 描述 默认值
op {Object} 主要参数
op.wizardId {String} 委托页面初始化的元素id "wizard"
op.btnContainer {String} 按钮的容器区域 "btn_container"
op.formContainer {String} 步骤的form表单容器区域 "form_step"
op.initJson {Array} 初始化Json []
op.items {Array} 步骤参数集合 []
op.items.stepId {String} 步骤的显示内容的标识ID
op.items.active {String} 是否为当前步骤
op.items.title {String} 步骤标题
op.items.prevFunction {Function} 上一步执行函数
op.items.nextFunction {Function} 下一步执行函数
op.wizardType {String} 步骤的初始类型，默认id "id"
op.successFunction {Function} 步骤向导提交成功后执行的函数
op.lastFunction {Function} 最后一步可以执行自己的函数，而不是执行默认提交
op.liEvent {Bool} li是否允许被点击 false
返回值:
{jQuery} jQuery对象
