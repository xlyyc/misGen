特殊说明
由于radio、checkbox、select在IE6底下外边框样式不生效，验证统一使用label包裹，然后往该父级元素添加“error_bg”，构建form表单的时候注意添加label元素进行包裹。
插件方法列表
validationEngine(p)
  • 表单验证
插件方法详细
{jQuery} validationEngine(p)
描述:
  • 所有的验证规则写在validate[]内，有多种验证，用逗号隔开，这里简要说明下常用的验证规则。 required：值不可以为空 length[0,100] ：文字允许长度 confirm[fieldID] ：匹配其他的表单元素的值，fieldID就是其他表单元素的id，这个主要用于再次确认密码 telephone ：数据格式要求符合电话格式 email ： 数据格式要求符合email 格式 onlyNumber ：只允许输入数字 noSpecialCaracters ：不允许出现特殊字符 onlyLetter ： 只能是字母 date ：必须符合日期格式YYYY-MM-DD
  • Ajax验证：{"jsonValidateReturn":["userName","ajaxUser","true"]} [0]:验证表单的ID [1]:验证方法 [2]:验证结果
参数列表:
参数名 类型 描述 默认值
p.allrules {Object} 配置规则，比如语言等等 规则支持任意扩展，扩展参加演示页面的用户名验证
p.validationEventTriggers {String} 表单验证时触发事件 focusout[keyup, blue]
p.promptPosition {String} 提醒位置,启用hideFormError模式的时候，这个参数控制没有实际意义 topRight [topLeft, topRight, bottomLeft, centerRight, bottomRight]
p.hideFormError {Bool} 隐藏原生的错误提示框，启用新模式 true
p.beforeSuccess {Function} 提交成功之前执行的函数 null

【验证类型】

       
<!-- 
注：验证规则均写在 validate[] 中，如有多条规则，用英文逗号(,)分割。 
例：validate[required,minSize[6],custom[onlyLetterNumber]] 
--> 
名称 示例 说明
required validate[required] 表示必填项
optional validate[optional] 表示可选项。若不输入，不要求必填，若有输入，则验证其是否符合要求。
length validate[length[2,3]] 字符串长度
limit validate[limit[2,3]] 数值区域
equals validate[equals[id]] 当前控件值需与 id 这个控件的值相同
minValue validate[minValue[1]] 最小值（该项为数字的最小值）
maxCheckbox[int] validate[maxCheckbox[2]] 最多选取的项目数（用于Checkbox）
minCheckbox validate[minCheckbox[2]] 最少选取的项目数（用于Checkbox）
qq validate[custom[qq]] 验证4-9位qq号码
phone validate[custom[phone]] 验证联系号码
mobilephone validate[custom[mobilephone]] 验证手机号码
telephone validate[custom[telephone]] 验证座机号码
email validate[custom[email]] 验证 Email 地址
domain validate[custom[domain]] 验证域名
zipcode validate[custom[zipcode]] 验证邮政编码
idCard validate[custom[idCard]] 验证身份证
mp3 validate[custom[mp3]] 验证mp3链接地址
notZero validate[custom[notZero]] 正整数
oneToNine validate[custom[oneToNine]] 1-9的整数
points validate[custom[points]] 1~999的整数
awardTimes validate[custom[awardTimes]] 1~99999的整数
date validate[custom[date]] 验证日期，格式为 YYYY-MM-DD
ip validate[custom[ip]] 验证 ipv4 地址
url validate[custom[url]] 验证 url 地址
onlyNumber validate[custom[onlyNumber]] 只接受填数字
onlyInteger validate[custom[onlyInteger]] 只接受填整数
onlyLetter validate[custom[onlyLetter]] 只接受填英文字母（大小写）
chinese validate[custom[chinese]] 只接受填中文
noSpecialCaracters validate[custom[noSpecialCaracters]] 只接受数字和英文字母
imageCaracters validate[custom[imageCaracters]] 只接受百分数或者像素值，例如15%或者15px
onlyFile validate[custom[onlyFile]] 只接受英文字母、数字和中文
noSpecialCaractersNew validate[custom[noSpecialCaractersNew]] 目录或者文件名的合法性
ajax validate[ajax[ajaxName]]

ajaxName方法名称
accept validate[accept[gif:png:jpg]]] 接受的文件类型
funcCall validate[funcCall[functionName]] 调用外部函数验证
